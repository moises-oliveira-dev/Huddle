# Huddle
Huddle é um site responsivo desenvolvido como prática de HTML e CSS,
com foco em layout moderno, organização de componentes e efeitos visuais.
O projeto segue um design limpo e foi criado para aprimorar habilidades
em front-end.
## Preview
![Preview do projeto](./design-20260108T224000Z-3-001/readme.png)
## Tecnologias utilizadas
- HTML5
- CSS3
- Flexbox
- Google Fonts (Poppins)
## Aprendizados
- Estruturação semântica em HTML
- Organização de CSS
- Uso de Flexbox para layout
- Estilização com variáveis CSS
- Efeitos hover em botões e ícones
